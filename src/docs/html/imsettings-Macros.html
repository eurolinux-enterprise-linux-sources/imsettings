IMSETTINGS_NONE_CONF

#define IMSETTINGS_NONE_CONF			"none"

The configuration filename used for "disabled". The real filename would be IMSETTINGS_NONE_CONF + something specified with --xinput-suffix build option that the default value is ".conf".
